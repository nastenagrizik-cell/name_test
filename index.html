Topline Generator Production V3
Боевой клиентский инструмент под GitHub + Render
Ориентир: вариант 3 + капучино-topline

Загрузите боевую базу и получите итоговый XLSX в логике капучино-шаблона с добавленным листом «Аудитория».

Инструмент рассчитан на monadic name test: фиксированные метрики, прямое сравнение, имиджевый блок, значимости по долям и автодобавление дополнительных вопросов через конфиг.

Ключевые листы: САММАРИ, полные таблицы, значимости, Аудитория.
Открытый вопрос на этапе V1 можно игнорировать через параметр ignoreOpenQuestionAtV1.
Чем больше названий в тесте, тем больше столбцов будет создано автоматически.

Что делает production V3

Распознает шкальные, single choice и multi-response блоки по заголовкам колонок.
Строит TOP-2 и полные распределения для основных метрик.
Считает доли по прямому сравнению и имиджевому блоку.
Формирует лист значимостей по z-test для долей и выводит метки A/B/C/D.
Строит обязательный лист «Аудитория» по ключевым профилям выборки.
