Topline generator

Загрузите Excel-базу, проверьте автоматически найденные блоки и сформируйте topline в Excel без ручной сборки таблиц.

Шаг 1

Загрузка базы

Сервис читает Excel, определяет стандартные вопросы, дополнительные метрики и готовит структуру topline.

Загрузите файл базы, чтобы начать.
Шаг 2

Стандартные вопросы

Вопросы сгруппированы компактно: видно тип блока и количество найденных названий, без длинных простыней заголовков.

Шаг 3

Имиджевые высказывания

Здесь можно быстро проверить, какие имиджевые формулировки нашлись автоматически и где может потребоваться ручная проверка.

Шаг 4

Дополнительные метрики

Похожие вопросы объединяются в группы. Название метрики можно задать один раз на всю группу, а не для каждого названия отдельно.

Шаг 5

Расчёт

После проверки блоков сервис сформирует Excel-файл со всеми листами topline.

Что изменилось

Интерфейс стал компактнее: вместо длинных текстов вопросов показываются короткие карточки по типам блоков и числу найденных названий. [file:256][file:257]

  • Стандартные вопросы — по группам.
  • Дополнительные метрики — с объединением похожих формулировок.
  • Имиджевые высказывания — отдельной проверкой.
  • Расчёт — одной кнопкой после быстрой проверки.

Как пользоваться

Сначала загрузите базу, затем проверьте компактные карточки в стандартных и дополнительных вопросах, после чего запускайте расчёт. [file:255][file:257]
