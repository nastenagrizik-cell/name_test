# Topline Generator v4

Обновлённая версия для GitHub и Render.

## Что изменилось

- главный файл теперь `index.html`;
- интерфейс упрощён, без технических блоков и конфигов на экране;
- кнопка `Проверить базу` показывает понятный диагностический статус;
- логика лучше работает с сырой базой и текстовыми/кодированными ответами;
- итоговая выгрузка сохраняет листы `САММАРИ`, `полные таблицы`, `значимости`, `Аудитория`.

## Структура репозитория

- `index.html`
- `assets/app.js`
- `render.yaml`
- `.gitignore`
- `README.md`
