# Topline Generator

Готовый репозиторий для деплоя на Render через GitHub.

## Что делает проект

Приложение принимает Excel-базу исследования и формирует итоговый `.xlsx` с листами:

- `САММАРИ`
- `Полные таблицы`
- `Значимости`
- `Аудитория`

## Стек

- Node.js
- Express
- Multer
- SheetJS (`xlsx`)
- ExcelJS

## Деплой на Render

### Вариант 1: через render.yaml

1. Создайте новый GitHub-репозиторий.
2. Загрузите в него все файлы из этого проекта.
3. На Render выберите `New +` → `Blueprint`.
4. Подключите репозиторий.
5. Render автоматически прочитает `render.yaml` и создаст web service.

### Вариант 2: вручную

- Runtime: `Node`
- Build Command: `npm install`
- Start Command: `npm start`

## Важные замечания

### 1. Это рабочая v1, но не финальный кастомный production-движок под каждый нюанс шаблона

В код уже заложены:
- чтение Excel-книги;
- выделение шкальных вопросов 1–5;
- расчет ТОП-2;
- сборка полных таблиц;
- расчет значимостей через z-test по долям;
- сборка аудитории по ключевым словам в названиях колонок.

### 2. Что почти наверняка захочется донастроить позже

- более точный маппинг вопросов в человекочитаемые заголовки;
- точное объединение монодических блоков по названиям тестируемых концептов;
- финальное Excel-оформление ближе к вашим текущим top-line шаблонам;
- исключение служебных колонок из summary;
- расширение под открытый вопрос.

## Логика текущей версии

- Один респондент = одна строка.
- Пропуски = пустые ячейки.
- Summary показывает ТОП-2 для распознанных шкал 1–5.
- Полные таблицы показывают ТОП-2 и ответы 1–5.
- Значимости считаются для ТОП-2 и для каждого варианта ответа между всеми распознанными столбцами внутри блока.
- Аудитория строится по полям, в названиях которых встречаются ключевые слова про пол, возраст, частоту, новинки, рестораны и посещение.
