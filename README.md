# Topline Generator Production V3

Боевой комплект файлов для GitHub и Render.

## Что это

Клиентский web-интерфейс для обработки базы исследования по тестам названий и выгрузки итогового XLSX в логике:
- `САММАРИ`;
- `полные таблицы`;
- `значимости`;
- `Аудитория`.

Архитектура ориентирована на **вариант 3** и шаблон **капучино-topline**.

## Что уже делает

- читает `.xlsx/.xls/.csv` в браузере;
- распознает названия концептов по заголовкам колонок;
- считает основные шкальные метрики 1–5 и TOP-2;
- считает прямое сравнение по favorite / first buy;
- считает имиджевые multi-response показатели;
- формирует лист значимостей на основе z-test для долей;
- добавляет лист `Аудитория`;
- позволяет через JSON подключать дополнительные вопросы.

## Структура проекта

- `topline-generator.html` — главный интерфейс;
- `assets/app.js` — боевой JS-движок;
- `assets/default-config.json` — конфиг V3;
- `render.yaml` — конфиг для Render Static Site;
- `.gitignore` — служебный файл.

## Как загрузить на GitHub

1. Создайте новый репозиторий.
2. Загрузите все файлы этой папки в корень репозитория.
3. Проверьте, что `topline-generator.html` лежит в корне.

## Как задеплоить на Render

1. Подключите GitHub-репозиторий в Render.
2. Выберите Static Site.
3. Build Command оставьте пустой.
4. Publish Directory: `.`

## Ограничения

Это боевой клиентский инструмент, но он зависит от структуры входной базы. Если в следующем проекте названия колонок будут сильно отличаться, нужно обновить `assets/default-config.json`.
