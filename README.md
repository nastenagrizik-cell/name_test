# Topline Naming Tool

Веб-интерфейс для автоматического формирования Excel-топлайна по тестам названий.

## Что делает

- Загрузка базы исследования (Excel).
- Автопоиск стандартных вопросов по ключевым фразам (не завязан на кодах).
- Подтверждение/отключение найденных блоков через интерфейс.
- Настройка дополнительных закрытых вопросов:
  - тип: шкала 1–5 (Top‑2) или single choice;
  - куда выводить: САММАРИ, полные таблицы, значимости.
- Автоматический расчет:
  - САММАРИ,
  - полные таблицы,
  - лист значимостей,
  - аудитория.

## Структура

- `index.html` — интерфейс.
- `app.js` — логика.
- (опционально) `render.yaml` — конфиг для Render.

## Локальный запуск

Откройте `index.html` в браузере (двойной клик или через любой статический сервер).

## Деплой на Render (Static Site)

1. Загрузите файлы в GitHub репозиторий.
2. В Render создайте **Static Site**:
   - подключите репозиторий;
   - Root Directory: `/`;
   - Build Command: `none`;
   - Publish Directory: `/`.
3. После первого билда откройте URL, который даст Render.
